Rekayasa Perangkat Lunak

Apa Itu Jurusan RPL? Rekayasa Perangkat Lunak atau yang biasa disingkat RPL adalah jurusan yang mempelajari dan mendalami pengembangan perangkat lunak, mulai dari pembuatan, pemeliharaan, manajemen organisasi pengembangan perangkat lunak, dan manajemen kualitas. Selain itu, jurusan ini juga berkaitan dengan software komputer, seperti pembuatan website, aplikasi, game, dan semua yang berhubungan dengan programmer. Intinya, jurusan RPL memiliki 3 hal penting yang akan dipelajari, yaitu coding, desain, dan algoritma. Pembelajaran tersebut bertujuan agar kamu bisa membuat program aplikasi berbasis mobile atau web. Di samping itu, kamu pun akan belajar tentang pengembangan sebuah sistem operasi dengan menggunakan struktur data yang efisien. Sebagai siswa jurusan RPL, kamu beruntung lho, Quipperian. Pasalnya, di zaman sekarang, industri sangat membutuhkan keberadaan tenaga teknisi di bidang pengembangan software.

Apa Saja yang Dipelajari di Jurusan RPL? Untuk meningkatkan kemampuanmu di dunia software, RPL memiliki beragam mata pelajaran, di antaranya adalah

1. Web Dinamis

Pelajaran Web Dinamis akan membimbingmu dalam pembuatan sebuah website. Pada pelajaran ini, kamu akan memahami teknik membuat website lewat berbagai macam aplikasi dan tentunya mulai mempelajari bahasa pemrograman.

2. Pemrograman Berorientasi Objek

Mata pelajaran yang biasa disingkat PBO ini membahas tentang penulisan coding yang berorientasi pada objek. Selain itu, kamu juga akan belajar berbagai konsep di dalam PBO seperti enkapsulasi, abstraksi, polimorfisme, dan konsep-konsep lainnya.

3. Pemrograman Desktop

Pada pembelajaran ini, kamu akan dibimbing untuk mengembangkan sebuah program yang dapat dijalankan oleh komputer. Di sini, kamu juga akan belajar macam-macam bahasa pemrograman desktop seperti Java, C, PHP, Visual Basic, Phyton, dan sebagainya.

4. Basis Data

Sebagai siswa jurusan RPL, kamu wajib banget mempelajari basis data. Sebab, basis data merupakan elemen dasar yang berfungsi untuk menyimpan seluruh data yang terdapat di dalam sebuah aplikasi.

5. Coding

Dalam pembelajaran coding, kamu akan diajak untuk memahami berbagai macam bahasa pemrograman, seperti pemrograman bahasa Pascal, Java, Web Server, PHP, HTML, CSS, dan sebagainya.

6. Algoritma

Pada pelajaran ini, kamu akan memahami langkah-langkah yang disusun secara berurutan untuk menyelesaikan masalah pemrograman. Beberapa materi yang terdapat pada pelajaran algoritma di antaranya adalah algoritma dasar dan algoritma tingkat lanjut.
